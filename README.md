        二、隐私政策
        1. 隐私保护原则
        本App严格遵循“隐私优先、本地存储、绝不泄露”的原则，尊重并保护用户的个人隐私及宝宝相关信息，所有用户数据均存储在用户个人设备本地，不上传至任何云端服务器、不与任何第三方共享，全力保障用户隐私安全。
        
        2. 数据收集与使用
        2.1 收集的数据类型
        本App仅在用户设备本地收集并存储以下数据，不进行任何远程收集或传输：
        • 用户主动输入的数据：包括宝宝姓名、性别、生日、出生时刻、头像、背景图、成长瞬间文案、照片、地点、分组标签、生长数据（身高、体重）、疫苗接种状态、评论等。
        • App运行产生的数据：包括数据创建时间、修改时间、备份文件等，仅用于App内部功能展示及数据管理，不用于任何其他用途。
        
        2.2 数据使用目的
        收集的数据仅用于实现本App的核心功能，具体包括：
        • 展示宝宝资料、成长瞬间、生长记录、疫苗接种信息，方便用户查看与管理。
        • 实现数据备份与恢复功能，帮助用户在换机、重装App时迁移数据。
        
        3. 数据存储与安全
        • 数据存储：所有用户数据均存储在用户个人设备的本地沙盒中，由iOS系统提供安全隔离保护，仅本App可访问，其他App及第三方无法获取。
        • 数据安全：本App不对用户数据进行加密存储（因数据仅在本地，且iOS系统已提供沙盒保护），用户应妥善保管个人设备，避免设备丢失、被盗或被他人非法访问，防止数据泄露。
        • 数据删除：用户可通过App内功能自行删除宝宝资料、成长记录、疫苗信息等数据，删除后的数据将无法恢复；卸载本App时，设备本地存储的所有相关数据将一同删除。
        
        4. 第三方服务说明
        • 本App不集成任何第三方SDK、广告服务、统计服务，不与任何第三方共享用户数据。
        • 本App可能会调用用户设备的相册权限（用于上传宝宝头像、背景图、成长瞬间照片），权限仅用于App内部功能，不收集、不传输相册内任何数据，用户可在设备设置中随时关闭该权限。
        
        5. 隐私政策更新
        开发者有权根据法律法规变化、产品功能更新等情况，对本隐私政策进行更新，更新后的政策将在App内显著位置公示，用户继续使用本App即视为同意更新后的隐私政策；若用户不同意更新后的政策，应立即停止使用本App。
